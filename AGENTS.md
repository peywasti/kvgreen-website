# Agent Guidelines for KV-Green Website

## Commands
- **Build**: `npm run build`
- **Dev server**: `npm run dev`
- **Lint**: `npm run lint`
- **Type check**: `npx tsc --noEmit`
- **Single test**: No test framework configured yet

## Code Style
- **Language**: TypeScript with strict mode
- **Framework**: Next.js 16, React 19
- **Styling**: Tailwind CSS v4
- **Imports**: Type imports first, then regular imports; relative paths for components
- **Components**: Functional with default export, PascalCase naming
- **Files**: kebab-case for pages/components, camelCase for utilities
- **Types**: Explicit typing for props, return types, and complex objects
- **Layout**: RTL for Persian content, responsive design
- **Error handling**: Use try/catch for async operations
- **Naming**: camelCase for variables/functions, PascalCase for components/types
- **Formatting**: No semicolons, 2-space indentation, single quotes for strings</content>
